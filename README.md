# CCNU宿舍电量统计
###寒假学习Python，练手专用！
==========================

 
####Python实现功能：
1. 自动抄表，利用反推的接口，抓取宿舍电表数据，并存入Sqlite
2. 对数据进行统计，并使用matplotlib画出曲线图

####解决问题：
1. 请求的时候使用playground发送json
2. 使用正则表达式进行数据解析
3. 操作sqlite数据库
4. 使用matplotlib进行绘图
5. 中文乱码问题（编码、中文字体）
6. 使用Jquery UI中的Tabs做前端展示
